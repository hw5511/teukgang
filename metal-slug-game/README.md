# Iron Rescue

A small browser-based run-and-gun arcade game inspired by classic side-scrolling action games. Built with plain HTML, CSS, JavaScript, and generated transparent PNG sprites.

## Play

- Move: `A/D` or arrow keys
- Jump: `Z` or `Space`
- Shoot: `X` or `J`
- Grenade: `C` or `K`
- Start/restart: `Enter`

## Local Run

```bash
python -m http.server 4173
```

Then open `http://localhost:4173`.
