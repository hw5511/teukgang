# Codex Shorts Workshop Deck

GitHub Pages deployment bundle for the Codex shorts automation workshop.

## Pages Entry

- `index.html`: clean presentation version with the thumbnail rail disabled.

## Source / Backup Files

- `codex-shorts-workshop.html`: original deck source with rail toggle.
- `codex-shorts-standalone-src.html`: standalone source used by the bundler.
- `standalone.html`: fully bundled standalone HTML.
- `deck-stage.js`: slide deck web component runtime.
- `shots/`: visual check screenshots from the source package.
- `scratchpad.md`: source notes.

## Controls

- Right arrow / space: next slide
- Left arrow: previous slide
- Number keys: jump to slide
- `R`: reset to first slide
