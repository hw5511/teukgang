8. Cloudflare Pages 배포

GitHub 없이도 진짜 URL을 만들 수 있습니다

Wrangler 로그인은 브라우저 OAuth 창에서 학생이 직접 승인합니다.

npx -y wrangler login

npx -y wrangler pages project create 프로젝트명 --production-branch main

npx -y wrangler pages deploy . --project-name 프로젝트명 --commit-dirty=true

브라우저에 Cloudflare 로그인 창이 뜨면 직접 로그인하고 승인합니다.

9. 결과 확인

배포는 URL 확인까지가 끝입니다

공개 URL에서 실제 게임 캔버스가 보이고 조작되는지 확인합니다.

  • 배포 URL 접속
  • 게임 화면 렌더링 확인
  • 시작 버튼 / 조작 테스트
  • 모바일 화면에서 터치 UI 확인
통합 프롬프트

실습 시간이 부족할 때는 이걸로 한 번에 진행합니다

HTML/CSS/JS만 사용해서 브라우저에서 실행되는 오리지널 2D 웹 게임을 만들어줘.

조건:
- 플레이어 이동, 점프, 아이템, 적, 점수, 게임오버, 클리어 화면을 넣어줘.
- 필요한 에셋은 배경을 투명화할 수 있게 만들어 assets 폴더에 저장하고 적용해줘.
- 모바일 반응형과 터치 조작 UI를 추가해줘.
- 로컬에서 실행해서 정상 작동을 확인해줘.

마지막으로 Cloudflare Pages에 배포해줘.
Wrangler CLI가 필요하면 설치하고, wrangler login을 실행해줘.
내가 로그인하면 Pages로 배포하고 URL을 확인해줘.
