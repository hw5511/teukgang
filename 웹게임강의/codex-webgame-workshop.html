Codex로 나만의 웹 게임 만들기

AI와 대화하며 2D 웹 게임을 만들고, 에셋과 모바일 UI를 더한 뒤 Cloudflare Pages로 바로 공유합니다.

결과물브라우저에서 실행되는 오리지널 2D 게임
에셋AI가 만든 캐릭터, 적, 아이템 이미지
모바일터치 버튼으로 플레이 가능한 반응형 UI
배포친구에게 공유할 수 있는 pages.dev 링크
전체 흐름

코딩 수업이 아니라, AI와 함께 결과물을 완성하는 수업

1설치와 로그인Codex 실행 후 GPT 계정으로 로그인
2게임 콘셉트장르, 캐릭터, 목표를 한 문장으로 정리
3첫 버전 제작HTML/CSS/JS 게임을 Codex가 생성
4로컬 테스트브라우저에서 직접 플레이하며 피드백
5에셋 적용투명 배경 PNG를 만들고 게임에 연결
6배포와 공유Cloudflare Pages로 URL 생성
STEP 1 · 첫 게임 만들기

처음부터 복잡하게 말하지 말고, 실행되는 게임부터 만듭니다

PROMPT첫 요청은 장르와 실행 환경을 분명하게
슈퍼마리오처럼 귀여운 횡스크롤 2D 웹 게임을 만들어줘.
HTML/CSS/JS만 사용해서 브라우저에서 바로 실행되게 해줘.

플레이어는 좌우 이동과 점프를 할 수 있고,
적과 아이템, 점수, 게임오버, 다시 시작 기능이 있으면 좋겠어.
Codex에서 바로 로컬 테스트할 수 있게 만들어줘.
STEP 2 · 로컬 테스트

만든 직후 바로 플레이하고, 느낀 점을 그대로 다시 말합니다

1Codex가 실행한 localhost 주소를 브라우저에서 열기
2이동, 점프, 충돌, 점수, 게임오버가 되는지 확인
3재미없는 부분을 자연어로 피드백하기
4버그가 보이면 스크린샷처럼 구체적으로 설명하기
TIP피드백은 짧고 구체적으로
점프가 너무 낮아서 답답해.
적이 너무 빨리 다가와서 처음부터 어렵게 느껴져.
점수를 얻을 때 더 확실한 효과가 있으면 좋겠어.
게임오버 후 다시 시작 버튼을 더 크게 보여줘.
STEP 3 · 에셋 만들기

도형으로 만든 게임에 캐릭터와 아이템을 입힙니다

PROMPT투명 배경 PNG라는 말을 꼭 넣기
이 게임에 사용할 캐릭터, 적, 아이템 이미지를 만들어줘.
게임에 바로 넣을 수 있게 투명 배경 PNG 에셋으로 만들어줘.

스타일은 귀엽고 밝은 도트 그래픽 느낌이면 좋겠어.
플레이어, 적, 코인, 회복 아이템을 각각 따로 쓸 수 있게 만들어줘.
STEP 4 · 완성도 높이기

게임은 한 번에 완성하지 않고, 플레이하며 계속 고칩니다

재미점프감, 속도, 적 등장 빈도, 아이템 보상 조정
피드백피격 효과, 점수 획득 효과, 화면 흔들림, 메시지 추가
목표스테이지 진행, 클리어 조건, 난이도 상승 추가
마무리시작 화면, 게임오버 화면, 다시 시작 흐름 정리
STEP 5 · 모바일 반응형

친구에게 링크를 보내려면 휴대폰에서도 플레이되어야 합니다

PROMPT터치 UI와 화면 크기 대응을 함께 요청
모바일에서도 플레이할 수 있게 반응형으로 개선해줘.
화면 크기에 맞게 Canvas와 UI가 조정되게 하고,
모바일에서는 좌우 이동, 점프, 공격을 터치 버튼으로 조작할 수 있게 해줘.

버튼은 엄지로 누르기 쉽게 화면 아래쪽에 배치해줘.
STEP 6 · Cloudflare 가입

배포 전에 수강생이 직접 가입하고 로그인합니다

1cloudflare.com 접속
2계정 생성 또는 기존 계정 로그인
3이메일 인증이 있으면 먼저 완료
4로그인된 브라우저 상태에서 Codex로 돌아오기
강사용 멘트

이 단계는 배포를 위한 준비입니다. Codex가 비밀번호를 대신 입력하지 않고, 로그인 승인은 여러분이 직접 합니다.

STEP 7 · Pages 배포

Codex에게 Wrangler 로그인을 실행하고 배포까지 맡깁니다

PROMPT수강생은 브라우저 인증만 직접 진행
wrangler cli를 설치해줘.
그다음 wrangler login을 실행해줘.

브라우저가 열리면 내가 직접 Cloudflare 로그인을 승인할게.
인증이 완료되면 이 게임을 Cloudflare Pages로 배포해줘.
배포가 끝나면 접속 가능한 URL을 알려줘.
문제 생겼을 때

막히면 해결도 Codex와 대화하면서 진행합니다

게임이 안 열림에러 메시지를 보여주고 로컬 서버를 다시 실행해달라고 요청
조작이 이상함어떤 키나 버튼에서 문제가 나는지 한 문장으로 설명
이미지가 안 보임파일 경로와 이미지 확장자를 확인해달라고 요청
배포 실패Wrangler 출력 마지막 부분을 기준으로 해결 요청
오늘의 제작 공식

아이디어를 말하고, 실행하고, 보고, 다시 부탁합니다

1만들고 싶은 것장르와 분위기를 말한다
2실행되는 첫 버전HTML/CSS/JS로 만든다
3보면서 피드백직접 플레이하고 고친다
4에셋 적용투명 배경 PNG를 넣는다
5모바일 대응터치 UI를 추가한다
6URL 공유Cloudflare Pages로 배포한다
마무리

오늘 만든 게임은 끝이 아니라 시작입니다

캐릭터를 바꾸고, 스테이지를 늘리고, 보스전을 추가하고, 다른 장르로 다시 만들 수 있습니다.
