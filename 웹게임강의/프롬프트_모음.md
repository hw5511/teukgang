# Codex 웹게임 특강 프롬프트 모음

## 1. 첫 게임 만들기

```text
슈퍼마리오처럼 귀여운 횡스크롤 2D 웹 게임을 만들어줘.
HTML/CSS/JS만 사용해서 브라우저에서 바로 실행되게 해줘.
플레이어는 좌우 이동과 점프를 할 수 있고, 적과 아이템, 점수, 게임오버, 다시 시작 기능이 있으면 좋겠어.
Codex에서 바로 로컬 테스트할 수 있게 만들어줘.
```

## 2. 에셋 생성 요청

```text
이 게임에 사용할 캐릭터, 적, 아이템 이미지를 만들어줘.
게임에 바로 넣을 수 있게 투명 배경 PNG 에셋으로 만들어줘.
스타일은 귀엽고 밝은 도트 그래픽 느낌이면 좋겠어.
```

## 3. 생성한 에셋 적용

```text
방금 만든 PNG 에셋들을 게임에 적용해줘.
플레이어, 적, 아이템이 각각 이미지로 보이게 하고,
기존 도형 그래픽은 이미지 기반으로 교체해줘.
```

## 4. 게임 완성도 높이기

```text
게임 완성도를 높여줘.
점수, 체력, 스테이지 진행감, 피격 효과, 아이템 획득 효과,
게임오버 화면과 다시 시작 흐름을 더 자연스럽게 만들어줘.
```

## 5. 모바일 반응형

```text
모바일에서도 플레이할 수 있게 반응형으로 개선해줘.
화면 크기에 맞게 Canvas와 UI가 조정되게 하고,
모바일에서는 좌우 이동, 점프, 공격을 터치 버튼으로 조작할 수 있게 해줘.
```

## 6. Cloudflare Pages 배포

```text
wrangler cli를 설치해줘.
그다음 wrangler login을 실행해줘.
브라우저가 열리면 내가 직접 Cloudflare 로그인을 승인할게.
인증이 완료되면 이 게임을 Cloudflare Pages로 배포해줘.
배포가 끝나면 접속 가능한 URL을 알려줘.
```

## 7. 수업용 통합 프롬프트

```text
나는 Codex를 처음 써보는 수강생이고, AI로 간단한 2D 웹 게임을 만들고 싶어.
귀엽고 밝은 분위기의 횡스크롤 게임을 HTML/CSS/JS로 만들어줘.

요구사항:
- 브라우저에서 바로 실행되는 정적 웹 게임
- 플레이어 이동, 점프, 적, 아이템, 점수, 게임오버, 다시 시작 기능
- 코드 구조는 이해하기 쉽게 파일을 나눠줘
- Codex 안에서 바로 로컬 테스트할 수 있게 해줘
- 이후 이미지 에셋을 적용하기 쉽게 만들어줘
```
