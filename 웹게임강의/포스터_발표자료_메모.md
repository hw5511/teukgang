# 포스터 및 발표자료 메모

## 포스터 핵심 문구

```text
Codex로 나만의 웹 게임 만들기

AI와 대화하며 2D 웹 게임을 만들고,
캐릭터와 아이템 에셋을 적용한 뒤
바로 공유 가능한 웹 URL까지 완성하는 1시간 실습
```

## 이런 걸 만들어봅니다

- 브라우저에서 실행되는 오리지널 2D 게임
- AI가 생성한 캐릭터, 적, 아이템 에셋
- 모바일에서도 플레이 가능한 터치 UI
- 친구에게 공유할 수 있는 웹 배포 링크

## 대상

- Codex를 처음 써보는 분
- AI로 게임 만들기를 경험해보고 싶은 분
- 코딩 경험이 많지 않아도 참여 가능

## 일정

- 7월 7일 16:00~17:00

## 선호한 포스터 방향

- 세로형 포스터
- 따뜻하고 귀여운 분위기
- 테이블 위 태블릿과 커피가 있는 장면
- 태블릿 화면 안에는 귀여운 마리오풍 2D 게임 화면
- 도트 그래픽 캐릭터 사용
- 우측 하단에는 QR 코드가 들어갈 빈 공간 확보
- 일정 영역은 필요에 따라 비워둘 수 있게 구성
