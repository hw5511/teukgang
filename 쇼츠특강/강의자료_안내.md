# 쇼츠특강 자료 안내

Codex로 AI 쇼츠를 자동화하는 특강용 실습 자료입니다.

## 먼저 볼 파일

- `course-prompts/student-prompts.md`: 수업 중 순서대로 투입할 최종 프롬프트 목록
- `standalone-html/쇼츠특강 (단독실행).html`: 단독 실행 가능한 수업용 HTML 자료
- `course-prompts/tests/p09-quality-audit.v1.result.md`: 최종 결과물 검수 리포트

## 최종 영상

- `out/image-dancing-plague-short.mp4`: 이미지 생성형 쇼츠 샘플
- `out/graphics-cycloid-short.mp4`: Remotion 그래픽형 쇼츠 샘플

## 실습 프로젝트

이 폴더는 Remotion + Typecast 실습 프로젝트를 포함합니다.

```powershell
npm install
npm run typecheck
npm run caption:review
npm run caption:review:graphics
```

Typecast API 키는 파일에 저장하지 말고 `TYPECAST_API_KEY` 환경변수로만 설정합니다.

## 포함된 주요 폴더

- `storyboards/`: 이미지형/그래픽형 storyboard JSON
- `assets/`: 이미지 프롬프트와 그래픽 storyboard 설명
- `public/`: Remotion에서 사용하는 이미지/오디오 에셋
- `src/`: Remotion composition 코드
- `tools/`: 자막 청크 검수용 로컬 웹 UI
- `course-prompts/`: 수업용 프롬프트와 테스트 로그
