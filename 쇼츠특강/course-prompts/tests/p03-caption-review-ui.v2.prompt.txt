방금 만든 P03 검수 UI와 storyboard JSON을 개선해줘.

현재 문제:
- subtitleChunks 중 일부가 어색하게 잘렸다.
- 예: "한 여자가 멈추지" / "못한 채 춤을" / "춤을 추기 시작했습니다"처럼 단어가 반복되거나, 의미 단위가 부자연스럽다.
- 수업에서는 이 문제가 매우 중요하다. 쇼츠 자막은 TTS가 읽는 순서와 정확히 맞아야 하고, 조사나 용언이 어색하게 찢기면 안 된다.

이번 단계에서 할 일:
1. storyboards/image-dancing-plague.storyboard.json의 subtitleChunks를 다시 정리해줘.
   - narration 문장의 단어 순서를 그대로 유지해.
   - chunk들을 순서대로 이어 붙였을 때 narration과 의미가 같아야 해.
   - 같은 단어가 중복되거나 빠지면 안 돼.
   - 한 chunk는 보통 2~4어절로 만들어.
   - 조사만 따로 떨어지거나, 용언이 이상하게 분리되지 않게 해.
   - "말이 안되는 / 것 같죠" 같은 분할은 금지야. "말이 안 되는 것 같죠"처럼 의미 단위가 살아야 해.
   - 시작초와 종료초는 씬 durationSeconds 안에 들어가야 하고, 앞 chunk 종료초가 다음 chunk 시작초와 자연스럽게 이어져야 해.

2. tools/caption-review.html에 검수 보조 기능을 추가해줘.
   - 각 씬마다 "청크를 이어 붙인 문장"을 보여줘.
   - narration과 chunk 연결문이 다르면 경고를 보여줘.
   - 중복 단어가 의심되거나 시간이 겹치면 경고를 보여줘.
   - 저장 버튼을 누르기 전에도 화면에서 문제를 바로 볼 수 있어야 해.

3. 서버는 기존처럼 저장 기능을 유지해줘.
   - 이번 단계에서도 이미지 생성, Typecast TTS 호출, 영상 렌더링은 하지 마.

4. 검증해줘.
   - JSON 파싱
   - npm run typecheck
   - subtitleChunks가 narration 순서를 보존하는지 간단한 스크립트나 수동 체크 결과를 마지막 응답에 적어줘.
