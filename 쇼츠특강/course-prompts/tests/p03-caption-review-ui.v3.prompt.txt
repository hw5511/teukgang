P03 자막 청크를 마지막으로 더 자연스럽게 다듬어줘.

현재 v2는 중복/누락은 없어졌지만, 아직 다음처럼 문법적으로 덜 자연스러운 청크가 남아 있어.
- "한 여자가 멈추지" / "못한 채 춤을"
- "사람들은 더 깊은" / "혼란 속으로 들어갔습니다."

이번에는 어절 수보다 "의미 단위"와 "문법 단위"를 우선해줘.

수정 대상:
- storyboards/image-dancing-plague.storyboard.json

청크 분할 규칙:
- chunk를 이어 붙이면 narration과 정확히 같아야 한다.
- 중복/누락 금지.
- 한 chunk는 보통 2~5어절까지 허용한다.
- 복합 서술어, 부정 표현, 관형어+명사는 되도록 같은 chunk에 둔다.
  - 좋은 예: "멈추지 못한 채"
  - 나쁜 예: "멈추지" / "못한 채"
  - 좋은 예: "더 깊은 혼란 속으로"
  - 나쁜 예: "더 깊은" / "혼란 속으로"
- 너무 짧은 조각이 연속으로 나오면 자막이 튀어 보이므로 피한다.
- TTS가 빠르게 읽는 쇼츠라고 생각하고, 한 화면에 보여도 자연스러운 자막 덩어리로 나눠라.

tools/caption-review.html의 검수 기능은 유지해줘.
이미지 생성, Typecast TTS 호출, 영상 렌더링은 하지 마.

검증:
- JSON 파싱
- npm run typecheck
- 모든 scene에서 chunks를 이어 붙인 문장이 narration과 정확히 같은지 확인

마지막 응답에는 수정한 청크 예시와 검증 결과만 짧게 정리해줘.
