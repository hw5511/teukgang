# graphics-cycloid storyboard

이 문서는 P06 기획안 `사이클로이드의 두 가지 반직관적 성질`을 그래픽형 쇼츠용 storyboard로 정리한 것입니다.

## 공통 정보

- 프로젝트명: `graphics-cycloid`
- 캔버스: `1080x1920`
- 프레임레이트: `30fps`
- 형식: `graphics` 씬 중심
- 검수 포인트: `headline`, `caption`, `narration`, `subtitleChunks`

## 씬 요약

### Scene 1. `brachistochrone-hook`

- 대본: 같은 출발점과 도착점에서 직선이 가장 빠르지 않다는 반전 소개
- 자막 청크:
  - 같은 출발점과 도착점인데,
  - 가장 빠른 길은
  - 가장 짧은 직선이 아닙니다.
  - 오히려 처음에 확 꺾여 내려가는
  - 곡선이 더 빨라집니다.
- 그래픽 요약:
  - 직선, 완만한 원호, 사이클로이드를 나란히 비교
  - 동시에 출발하는 발광 점 3개
  - 사이클로이드가 먼저 도착하도록 타이머 표시

### Scene 2. `brachistochrone-definition`

- 대본: 사이클로이드와 브라키스토크론의 이름을 정확히 소개
- 자막 청크:
  - 이 문제의 정답은
  - 사이클로이드이고, 중력 아래에서
  - 가장 빨리 도착하는 경로를
  - 브라키스토크론이라고 부릅니다.
- 그래픽 요약:
  - 길이와 시간을 분리해서 비교
  - 직선은 짧아도 늦고, 사이클로이드는 길어도 빠른 구조
  - 경과 시간과 길이 눈금을 함께 보여줌

### Scene 3. `cycloid-generation`

- 대본: 사이클로이드가 어떻게 생성되는지 시각화
- 자막 청크:
  - 이 곡선은 원이
  - 미끄러지지 않고 굴러갈 때
  - 바깥의 한 점이
  - 그리는 자취입니다.
- 그래픽 요약:
  - 원의 중심 이동과 회전을 동시에 애니메이션
  - 원둘레의 점이 그리는 궤적을 누적
  - 사이클로이드 생성 과정을 직접 보여줌

### Scene 4. `tautochrone`

- 대본: 같은 사이클로이드에서 출발점이 달라도 도착 시간이 같다는 타우토크론 성질 설명
- 자막 청크:
  - 더 놀라운 건,
  - 같은 사이클로이드에서는
  - 출발 위치가 달라도
  - 아래쪽까지 걸리는 시간이
  - 같다는 점입니다.
- 그래픽 요약:
  - 서로 다른 시작점 3개
  - 각 구슬이 동시에 출발
  - 도착점에서 동시에 도착하도록 동기화
  - 도착 순간 플래시로 결론 강조

### Scene 5. `summary`

- 대본: 전체 결론 정리
- 자막 청크:
  - 가장 빠른 길은,
  - 가장 짧은 길이 아니라
  - 속도를 먼저
  - 벌어두는 길입니다.
- 그래픽 요약:
  - 텍스트 중심의 엔딩 카드
  - 사이클로이드 곡선은 배경에서 은은하게 유지
  - 마지막 한 줄로 메시지를 고정

## 검수 흐름

1. `caption:review:graphics`로 검수 서버를 실행한다.
2. 각 씬의 `subtitleChunks`를 `" "`로 이어 붙인 뒤, 여러 공백을 하나로 정규화해 `narration`과 비교한다.
3. `normalize(chunks.join(" ")) === normalize(narration)`인지 확인한다.
4. 수정 후 `Save JSON`으로 storyboard를 다시 저장한다.
