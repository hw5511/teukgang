# Codex 웹게임 특강 슬라이드

GitHub Pages로 배포하는 `Codex로 나만의 웹 게임 만들기` 수업용 HTML 슬라이드입니다.

## 구성

- `index.html` - 배포용 슬라이드 엔트리
- `deck-stage.js` - 슬라이드 스테이지와 왼쪽 슬라이드 목록 UI
- `uploads/수업_순서별_핵심_프롬프트.md` - 수업 순서별 상세 프롬프트 원문

## 로컬 확인

```bash
python -m http.server 4173
```

브라우저에서 `http://localhost:4173`을 엽니다.
