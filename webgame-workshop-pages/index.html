첫 프롬프트 — 그대로 복사하세요

프롬프트콘셉트 한 줄은 바꿔 써도 됩니다
방금 정한 콘셉트로 웹 게임 첫 버전을 만들어줘. · HTML / CSS / JavaScript만 사용 (브라우저에서 바로 실행) · 파일은 index.html · style.css · script.js로 나눠줘 · 원작 캐릭터를 따라 하지 않는 오리지널 게임으로 · 플레이어 이동 · 점수(아이템) · 적/장애물 · 게임오버 · 다시 시작 · 코드는 초보자가 읽어도 이해하기 쉽게 완성 후 로컬에서 바로 테스트할 수 있게 실행 방법까지 알려줘.
